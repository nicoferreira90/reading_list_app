About the App

MyReadingList is a simple Django app designed to help users track the books they've read. I built it primarily to experiment with modern server-side JavaScript technologies like HTMX and Alpine.js. The goal was to create a user experience similar to a single-page application, but with less complexity than a full-fledged JavaScript framework like React or Vue.js.
